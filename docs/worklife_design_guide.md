### 🧩 컴포넌트 가이드

#### 버튼 (Button)

- **Solid 버튼**

  - Primary: 네이비 배경 + 흰색 텍스트
  - Secondary: 민트 배경 + 네이비 텍스트
  - Disabled: 밝은 회색 배경 + 회색 텍스트 (투명도 60%)

- **Outline 버튼**
  - 테두리: 1px solid `#65C3BA`, 텍스트 민트색, 호버 시 배경 민트로 채움.

#### 알럿 / 공지 (Alert / Notice)

| 유형 | 배경색    | 텍스트색  | 예시                                   |
| ---- | --------- | --------- | -------------------------------------- |
| 성공 | `#E8F5E9` | `#2E7D32` | “저장이 완료되었습니다.”               |
| 경고 | `#FFF3E0` | `#F57C00` | “변경사항이 아직 적용되지 않았습니다.” |
| 오류 | `#FFEBEE` | `#C62828` | “데이터 로드에 실패했습니다.”          |

#### 라디오 & 체크박스

- 활성 색상: 민트 `#65C3BA`
- 비활성: 밝은 회색 `#E5E7EB`
- 호버: 스카이 블루 `#A6E3E9`

### 🧠 UI 무드보드

**전체 느낌:** Notion과 Linear에서 영감을 받은 깔끔하고 차분한 업무형 디자인.

- 배경색: 화이트 또는 아주 연한 회색 (`#FAFAFA`)
- 코너 반경: 8px (버튼, 카드 등)
- 그림자: `0 2px 8px rgba(0, 0, 0, 0.08)`
- 트랜지션: 200~300ms ease-in-out (호버, 모달 애니메이션 등)
